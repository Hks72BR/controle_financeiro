## 🌐 Como Usar com Servidor (Para Instalar no Celular)

### Método Mais Fácil (Recomendado):

1. **Dê duplo clique em:** `1-INSTALAR.bat`
   - Isso instalará o servidor Python
   - Precisa fazer apenas uma vez!

2. **Depois, sempre que quiser usar:**
   - Dê duplo clique em: `2-RODAR_SERVIDOR.bat`
   - Acesse: `http://localhost:8000`

3. **No celular (mesma rede Wi-Fi):**
   - No PowerShell do PC, digite: `ipconfig`
   - Procure por "Endereço IPv4"
   - Acesse no celular: `http://SEU_IP:8000`

### Método Manual (Se preferir):

1. **Instalar uma única vez:**
   ```powershell
   pip install http-server
   ```

2. **Depois, para rodar:**
   ```powershell
   cd "C:\Users\higor\Desktop\controle financeiro"
   p
   ```

3. **Acesse:** `http://localhost:8000`

## 📲 Instalar no Celular/Desktop

### Chrome (Android/Desktop):
1. Abra o app
2. Procure o ícone ➕ ou ⋮
3. Clique em "Adicionar à tela inicial" ou "Instalar"

### Safari (iPhone):
1. Abra o app
2. Toque no ícone de compartilhar ⬆️
3. Role e toque em "Adicionar à Tela Inicial"
